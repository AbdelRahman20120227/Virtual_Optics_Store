- This was my Greatest experement.
- This is an android app that lets you fit sunglasses while being home
  using android front camera.


TECHNOLOGIES USED:

- OPENGLES.
- OPENCV.
- DLIB.

CHALLENGES INSIDE:

- Parsing 3d models.
- Facial features detection with high performance.
- HEAD POSE ESTIMATION.
- USING OPENGL efficently to draw 3d models on face.
- THREADING AND SYNCROUNIZATION. 

All important work is located in the package "main.java.com.example.ahmed.opengl".
